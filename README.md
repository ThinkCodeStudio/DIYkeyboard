## 硬件
* AT32F435ZMT7
* Flash W25Q128J
* SD卡
* 74HC165
* ST7789
* USB 3.0 HUB GL3520
* 编码器
* 拨码管 12pin*2
* 磁连接触点 (SPI&uart， 四个方向)
* 电池管理电路
* RTC电路
* RGB灯
* usb type-c (偏右侧靠近芯片)
* 蓝牙模块 (暂时没有选型)
