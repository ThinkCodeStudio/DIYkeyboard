# DIYkeyboard



# MCU

## esp32			

* 集成蓝牙

  

* 性能溢出

* 贵

* 功耗高

* 没有USB

* 有没用的功能 

* GPIO少
## atmega32u4
* 生态完善

* 有USB

  

* 性能太低

* 贵

* GPIO少
## STM32L433
* 低功耗

* 有USB

* 性能优

  

* 贵

* GPIO少


## CH583 risc-v  
* 低功耗

* 集成蓝牙

* 可编程

* 有USB

* 性能优

  

* 新芯片，生态差

* QFN封装 焊接困难

* 40 GPIO

## CH32V208WBU6 ON
* QFN68封装
* 53 GPIO



* 功能过于强大
* 没有低功耗

## STM32F407ZGxx

* LQFP 144
* 资料全
* 生态好
* 适配microPython
* 方便开发
* 功能强大



* 功耗30~40mA
* 贵
* 可能GPIO过多
* 不集成蓝牙

**注意功耗计算**
